Get Started

Speed in Software Development

Using Redux Cool toolkit, you will build Redux State Management Logic in our app a few times faster than if you use any other toolkit existing up to this date.

Intuitively Understandable

In the Redux Cool, the reducer functions are created with the help of Reducer-Tree which enables us to intuitively understand the entire redux logic at first glance.

Simple API

The Redux Cool API surface is very tiny. It has only two Top-Level Exports:
reducersCreator and actionsCreator

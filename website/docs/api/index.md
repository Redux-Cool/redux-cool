The Redux Cool API surface is very tiny. It has only two Top-Level Exports

- [reducersCreator](/docs/api/reducersCreator)
- [actionsCreator](/docs/api/actionsCreator)

## Importing

#### ES6

```javascript
import {
    reducersCreator,
    actionsCreator
} from "redux-cool"
```


#### CommonJS

```javascript
const {
    reducersCreator,
    actionsCreator
} = require("redux-cool")
```
